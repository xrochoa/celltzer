# CELLTZER 
***

## 1. Use box > line > cells

### Example 1:

```html
   <div class="box">
      <div class="line">
        <div class="cell xs-50"></div>
        <div class="cell xs-50"></div>
      </div>
    </div>
```

## 2. Responsive design:
+ xs for smartphone and above
+ sm for ipad and above
+ md for medium desktop
+ sm for large desktop and above


### Example 2:

```html
   <div class="box">
      <div class="line">
        <div class="cell xs-100 md-33"></div>
        <div class="cell xs-100 md-33"></div>
        <div class="cell xs-100 md-33"></div>
      </div>
    </div>
```
